METODOLOGIA

A metodologia adotada para o desenvolvimento da carteira de vacinação digital envolveu uma abordagem cuidadosa e estruturada, combinando a linguagem de programação C# no ambiente de desenvolvimento Visual Studio e o banco de dados Access. Além disso, foram realizadas pesquisas detalhadas para embasar as decisões de design e desenvolvimento do sistema.
